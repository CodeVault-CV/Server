## Features
- 스터디 생성시 스터디장의 깃허브 저장소와 연동
- 스터디 진행의 단위인 ‘세션’을 생성할 수 있음
- 세션 안에서 진행할 문제들을 생성할 수 있음
- 문제에 대한 풀이를 제출하면 깃허브 저장소에 자동으로 풀이가 업로드 됨
- 제출된 풀이를 스터디원들이 확인할 수 있고, 풀이에 대한 리뷰 작성 가능
